# CVE, CWE, NVD, and CVSS

This project explains how CVEs, CWEs, the National Vulnerability Database, and CVSS scores help organizations identify, classify, assess, and prioritize cybersecurity vulnerabilities.

## Topics Covered

- Purpose of CVE
- Severity and CVSS scoring
- CVE List and CVE Numbering Authorities
- Using CVEs and CVSS in vulnerability management
- Calculating and interpreting CVSS scores
